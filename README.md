## Project Overview

This project focuses on analyzing **bank marketing campaigns** to uncover insights such as:

- Campaign performance across different customer segments  
- Customer engagement with products and campaigns  
- Conversion rates and effectiveness of marketing strategies  
- Sentiment analysis of customer reviews for better decision-making  

The dashboard was initially created in **Power BI**, and a **Streamlit app** was built to showcase it interactively, making it easy for users to explore the data and visualize results.

---

## Features

- **Landing Page** with looping video demo of the Power BI dashboard  
- **Interactive Dashboard Screenshots** displayed in a clean two-column layout  
- **Downloadable PDF** of the full dashboard  
- **Data Collection Tab** with explanations of data sources  
- **Data Explorer** allowing users to preview and interact with CSV datasets  
- **Sentiment Analysis** performed on customer reviews using Python

## Datasets

The project uses **synthetic banking data** enriched with real-world locations in Trinidad and Tobago. The following datasets are included:

| Dataset | Description |
|---------|-------------|
| `data/banking_campaigns.csv` | Campaign information and details |
| `data/banking_engagement_data.csv` | Customer engagement metrics |
| `data/banking_products.csv` | Banking products offered |
| `data/customer_journey_data.csv` | Customer actions during campaigns |
| `data/customer_reviews_sentiment.csv` | Sentiment scores for customer reviews |
| `data/customer_reviews.csv` | Original customer reviews |
| `data/synthetic_bank_customers.csv` | Synthetic customer demographic data |
| `data/trinidad_and_tobago_all_cities_towns.csv` | Cities and towns in Trinidad and Tobago |
